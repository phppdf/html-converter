---
title: "html-converter documentation"
---

html-converter is a PHP library that converts HTML and CSS into PDF documents. It builds on top of [phppdf/phppdf](https://github.com/phppdf/phppdf), returning a `PdfDocumentBuilder` that you can further customise before serialising.

## Requirements

| Requirement | Version |
|---|---|
| PHP | 8.4+ |
| ext-dom | any |
| ext-libxml | any |

## Installation

```bash
composer require phppdf/html-converter
```
